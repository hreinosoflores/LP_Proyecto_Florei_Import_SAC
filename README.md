# LP_Proyecto_Florei_Import_SAC
Aplicación de Escritorio basado de Java Swing con conexión directa a base de datos MySql.

## Para ejecutar el proyecto
1. Usar un IDE tal como Eclipse. Configurar un espacio de trabajo con java 1.8.
1. Clonar el repositorio en el espacio de trabajo
1. Abrir y ejecutar **sql/BD_FloreiImport.sql** en un servidor Mysql local
1. Ejecutar **src/vistas/FrmPrincipal.java** como Java Application

Si se desea cambiar la configuracion de bd entrar a **/src/utils/MySQLConexion.java**
